# gestione-eventi-provaai

App Streamlit modulare per la gestione semplificata di eventi, artisti, format e staff per un'agenzia di spettacoli.

Istruzioni rapide:

1. Clona il repo
2. Crea un virtualenv e installa dipendenze: `pip install -r requirements.txt`
3. Avvia l'app: `streamlit run run_app.py`
